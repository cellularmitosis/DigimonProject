# Digimon-Character-Finder

## Description
Digimon: Character Finder lets you find a Digimon and their level. Type and search for a Digimon. You can also add your favorite Digimon to the Favorites tab or remove them.

### Highlights
- Utilized UIKit framework to build the app
- Persisted data through UserDefaults
- Implemented Swift Concurrency to make an API call and fetch JSON

You can download my [Digimon: Character Finder App](https://apps.apple.com/us/app/digimon-character-finder/id6449167225) on the App Store!
